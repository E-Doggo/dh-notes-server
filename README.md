## Description

This project has been made on [NestJS](https://github.com/nestjs/nest) V10.

## Project setup

<p>Please clone the repository to your local machine to proceed.</p>

```bash
$ git clone https://github.com/E-Doggo/dh-notes-server.git
```

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Resources
